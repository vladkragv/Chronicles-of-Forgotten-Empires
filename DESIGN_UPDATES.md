### 5. **Уникальные анимации (8 типов)**
Все блоки на странице имеют специальные анимации при появлении:

- **mysticalBounce** - Элемент появляется с мистическим отскоком
- **slideInWithGlow** - Скольжение слева со свечением
- **fadeInWithRotation** - Появление с 3D поворотом
- **expandFromCenter** - Расширение из центра страницы
- **enigmaticReveal** - Таинственное раскрытие с blur эффектом
- **arcaneAppear** - Архаичное появление с вращением
- **glitchAppear** - Глитч-эффект появления (как в Witcher)
- **fadeInWithMist** - Появление с туманом/blur эффектом

Все анимации настроены для высокой производительности с Intersection Observer API.

### 6. **Система редактирования блоков правил в админке**
Полностью новая система управления блоками правил!

**Доступ к админке:**
- URL: `http://localhost/project/admin.html` (или на вашем сервере)
- Логин: `admin-vladkragv`
- Пароль: `admin-vladkragv`

**Редактирование блоков в админке:**
1. Перейдите на вкладку "Правила"
2. Выберите подвкладку "🎮 Блоки кратких правил"
3. Каждый блок можно редактировать отдельно:
   - **Иконка/Эмодзи** - символ, который отображается в блоке
   - **Заголовок** - название блока (например, "Начало игры")
   - **Текст** - описание (например, "Каждый вытягивает героя")
   - **Фоновое изображение (URL)** - опционально, можно добавить изображение вместо эмодзи

**Добавление фонового изображения:**
- В поле "Фоновое изображение (URL)" вставьте ссылку на изображение
- Должен быть абсолютный URL (начинающийся с http:// или https://)
- Рекомендуемое разрешение: 100x100px или квадратное

**Добавление нового блока:**
- Нажмите кнопку "➕ Добавить новый блок"
- Заполните все поля
- Нажмите "💾 Сохранить блок"

## ⚙️ Технические детали

### Использованные API:
- Intersection Observer API - для оптимизированных scroll анимаций
- localStorage API - для сохранения данных блоков правил
- HTML5 Video API - для видеофона
- CSS Grid & Flexbox - для макета
- CSS Animations & Transitions - для анимаций

### Производительность:
- Все анимации используют GPU-ускоренные свойства (transform, opacity)
- Scroll события используют `{ passive: true }` для неблокирующего поведения
- Intersection Observer с threshold 0.1 для эффективного отслеживания видимости

### Совместимость:
- Chrome/Edge 60+
- Firefox 55+
- Safari 12.1+
- Требуется современный браузер с поддержкой ES6
