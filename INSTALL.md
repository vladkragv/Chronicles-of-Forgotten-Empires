# Инструкции по запуску сайта Катакомбы

## Быстрый старт

### Способ 1: Просто откройте index.html
1. Перейдите в папку `Site`
2. Дважды щелкните на `index.html`
3. Сайт откроется в браузере по умолчанию

### Способ 2: Используйте Live Server (VS Code)
1. Откройте папку `Site` в VS Code
2. Установите расширение "Live Server" (автор: Ritwick Dey)
3. Правый клик на `index.html` → "Open with Live Server"
4. Сайт откроется на `http://localhost:5500`

### Способ 3: Используйте Python
```bash
# Python 3.x
python -m http.server 8000

# или Python 2.x
python -m SimpleHTTPServer 8000
```
Затем откройте `http://localhost:8000` в браузере.

### Способ 4: Используйте Node.js
```bash
# Установите http-server глобально
npm install -g http-server

# Запустите в папке проекта
http-server
```

## Что делать, если страница не загружает карты?

### 1. Проверьте консоль браузера (F12)
- Откройте DevTools (F12 или Ctrl+Shift+I)
- Перейдите на вкладку "Console"
- Если видите ошибки, сообщите о них

### 2. Проверьте пути к изображениям
- Убедитесь, что папка `images/cards/` содержит изображения
- Имена файлов должны совпадать с тем, что в `js/data.js`

### 3. Если изображений нет
- Сайт всё равно работает! Будут показаны placeholder-изображения
- Добавьте реальные JPG/PNG с корректными названиями в `images/cards/`

## Как добавить собственные карты?

### 1. Добавьте изображение карты
```
images/cards/my-card.jpg  (180x250px рекомендуется)
```

### 2. Добавьте данные карты в js/data.js
```javascript
{
    id: 100,                           // Уникальный ID
    name: 'Название карты',            // Имя карты
    type: 'heroes',                    // Тип: heroes, spells, artifacts и т.д.
    atk: 3,                            // Атака (0 для артефактов/заклинаний)
    hp: 8,                             // ХР (0 если не требуется)
    mana: 2,                           // Стоимость МР
    ability: 'Описание способности',   // Текст способности
    image: 'my-card.jpg'               // Имя файла изображения
}
```

## Как добавить гибридную карту?

```javascript
{
    id: 101,
    name: 'Гибридная карта',
    type: 'beasts',                    // Основной тип
    hybridTypes: ['demons'],           // Дополнительные типы
    atk: 4,
    hp: 4,
    mana: 3,
    ability: 'Описание. [ГИБРИДНАЯ: Зверь-Демон]',
    image: 'hybrid-card.jpg',
    hybrid: true                       // Обязательный флаг!
}
```

## Развёртывание на сервер

### Вариант 1: GitHub Pages (бесплатно)
1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `/root`
5. Сайт будет доступен по адресу `https://yourusername.github.io/repository-name`

### Вариант 2: Netlify (бесплатно)
1. Перейдите на netlify.com
2. Нажмите "New site from Git" или "Deploy manually"
3. Загрузите папку `Site`
4. Сайт будет развёрнут на Netlify

### Вариант 3: Вконтакте + облако
1. Создайте облако вроде Яндекс.Диска, OneDrive
2. Загрузите все файлы
3. Включите публичный доступ
4. Поделитесь ссылкой

Если много пользователей, используйте CDN для доставки файлов

Если возникли проблемы:
1. Проверьте консоль браузера (F12)
2. Убедитесь, что все файлы на месте
3. Попробуйте другой браузер
4. Очистьте кэш (Ctrl+Shift+Delete)
5. Перезагрузите страницу (Ctrl+F5)
