# Катакомбы - Настольная Карточная Игра (Сайт)

Полнофункциональный веб-сайт для настольной карточной игры "Катакомбы".

## Структура проекта

```
Site/
├── index.html              # Главная страница
├── rules.html              # Страница с правилами
├── cards.html              # Каталог карт
├── about.html              # Страница "О игре"
├── css/
│   └── styles.css          # Основные стили (фэнтезийная тема)
├── js/
│   ├── main.js             # Основная функциональность
│   ├── data.js             # База данных карт (80+ карт)
│   ├── cards.js            # Функциональность страницы карт
│   └── rules.js            # Функциональность страницы правил
├── data/
│   └── cards.json          # (опционально) JSON база карт
├── images/
│   ├── logo.png            # Логотип (создать)
│   ├── banner.jpg          # Баннер главной (создать)
│   ├── cards/              # Папка с изображениями карт
│   └── icons/              # Иконки механик
└── pdf/
    ├── quick-rules.pdf     # Краткие правила (создать)
    └── full-rules.pdf      # Полные правила (создать)
```

### 2. Главная страница
- Анимированный баннер с названием игры
- Кнопка скачивания правил (PDF)
- Описание игры
- Галерея с 6 избранными картами
- Статистика (12 героев, 80+ карт, 2-4 игроков, 30-60 минут)

### 3. Раздел "Правила"
- **Краткие правила:**
  - Блок-схема игрового процесса
  - 6 ключевых механик с иконками
  - Кнопка скачивания PDF

- **Подробные правила:**
  - Оглавление с якорными ссылками
  - 8 основных разделов

  - Кнопка скачивания полной инструкции

### 4. Раздел "Карты"

- Модальное окно для просмотра полной информации о карте

- Поддержка гибридных карт (Зверь-Демон, Демон-Дух и т.д.)

- Заклинания (21)

### 5. Раздел "О игре"

## Установка и использование

1. **Клонируйте или откройте файлы в папке `Site/`**

2. **Откройте `index.html` в веб-браузере** (или используйте Live Server)

3. **Добавьте изображения:**
   - Поместите карточки в `images/cards/` (либо используются плейсхолдеры)
   - Используйте имена из `data.js` (например, `hero-1.jpg`, `spell-1.jpg`)

4. **Создайте PDF файлы:**
   - `pdf/quick-rules.pdf` - краткие правила
   - `pdf/full-rules.pdf` - полные правила

## Расширение

### Как добавить новые карты?
1. Откройте `js/data.js`
2. В массив `CARDS_DATABASE` добавьте объект:
```javascript
{
    id: 83,
    name: 'Имя карты',
    type: 'heroes',              // Тип
    atk: 3,                       // Атака
    hp: 8,                        // ХР
    mana: 0,                      // Стоимость МР
    ability: 'Описание способности',
    image: 'hero-13.jpg'          // Имя файла
}
```

### Как добавить гибридную карту?
```javascript
{
    id: 20,
    name: 'Химера',
    type: 'beasts',
    hybridTypes: ['demons'],      // Это ГИБРИДНАЯ карта
    atk: 4,
    hp: 4,
    mana: 3,
    ability: '...',
    image: 'hybrid-1.jpg',
    hybrid: true                  // Флаг гибридной карты
}
```
