## 📝 ПОШАГОВОЕ РУКОВОДСТВО

### Шаг 1: Открыть админку
```
Зайти на: https://ваш-сайт.com/admin.html
Логин: admin-vladkragv
Пароль: admin-vladkragv
```

### Шаг 2: Изменить основную информацию
1. Нажать на вкладку **"Главная"**
2. Изменить название, подзаголовок, описание
3. Нажать **"💾 Сохранить"**
4. Содержимое автоматически обновится на главной странице

### Шаг 3: Отредактировать карты
1. Нажать на вкладку **"Карты"**
2. Найти карту через поиск (опционально)
3. Нажать **"✏️ Редактировать"** на нужной карте
4. Изменить характеристики (название, АТК, ХР, МР, способность)
5. Загрузить новое изображение (указать URL или название файла)
6. Нажать **"💾 Сохранить карту"**
7. Карта обновится на странице "Игровые карты"

### Шаг 4: Добавить новую карту
1. Нажать **"➕ Добавить новую карту"**
2. Заполнить все поля
3. Нажать **"💾 Сохранить карту"**
4. Карта появится в каталоге

### Шаг 5: Поменять цвета
1. Нажать на вкладку **"Настройки"**
2. Найти нужный тип карты
3. Нажать на цветное поле и выбрать новый цвет
4. Нажать **"💾 Сохранить цвета"**
5. Цвета обновятся по всему сайту

## 📂 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ

### Структура файлов
```
js/
├── data.js              # База данных карт
├── storage.js           # Управление localStorage
├── dynamic-content.js   # Загрузка контента из хранилища
├── animations.js        # Анимации при скроллинге
├── main.js             # Основная функциональность
├── cards.js            # Логика страницы карт
├── rules.js            # Логика страницы правил
└── admin.js            # Логика админки

css/
├── styles.css          # Основные стили
├── fantasy.css         # Фэнтезийный дизайн
└── admin.css           # Стили админки

html/
├── index.html          # Главная страница
├── rules.html          # Правила
├── cards.html          # Каталог карт
├── about.html          # О игре
└── admin.html          # Админка
```

### Как работает localStorage

1. **При первом посещении**: Загружаются стандартные данные из `data.js` в localStorage
2. **При редактировании**: Новые данные сохраняются в localStorage
3. **При обновлении страницы**: Данные загружаются из localStorage
4. **При сбросе**: Все данные восстанавливаются на стандартные значения

### Экспорт/Импорт

Любые изменения можно сохранить или восстановить:
- **Экспорт**: Админка → Настройки → Скачать резервную копию
- **Импорт**: Админка → Настройки → Загрузить резервную копию

## 🎨 НОВЫЕ ФУНКЦИИ ДИЗАЙНА

### Фэнтезийная цветовая схема
- Золотые (#d49a3d) и коричневые (#2d241f) основные цвета
- Зелёные акценты (#6ba547) для природных элементов
- Тёмный красный (#8b3a3a) для акцентов

### Боковые заросли
- Автоматически отображаются по бокам каждой страницы
- Дополняют фэнтезийную атмосферу
- Адаптируются под разные разрешения экрана

### Кастомный скроллбар
- Золотой градиент цвета
- Плавные переходы и анимации
- Работает во всех современных браузерах

### Анимации при скроллинге
- Элементы появляются с плавной анимацией при прокрутке
- Карты масштабируются и светятся при приближении
- Текст плывёт снизу вверх
