# Quick Start Guide - Быстрый старт

## 🚀 Начало работы (5 минут)

### 1️⃣ Откройте админку
```
Адрес: admin.html
Логин: admin-vladkragv
Пароль: admin-vladkragv
```

### 2️⃣ Отредактируйте блоки правил
1. Перейдите на вкладку "Правила" → "Блоки кратких правил"
2. Нажмите "✏️ Редактировать" на любом блоке
3. Измените иконку, заголовок, текст
4. (Опционально) Добавьте URL фонового изображения
5. Нажмите "💾 Сохранить блок"

### 3️⃣ Добавьте видеофон
1. Подготовьте видео в формате MP4 (1920x1080)
2. Поместите файл: `/images/hero-background.mp4`
3. Откройте index.html - видео должно воспроизводиться

### 4️⃣ Посмотрите на результаты
- Откройте `index.html` - главная с видео
- Откройте `rules.html` - блоки правил с вашими изменениями
- Скролльте вниз - смотрите на красивые анимации

## 🎮 Основные функции

### Админка (admin.html)
| Вкладка | Что можно делать |
|---------|------------------|
| **Главная** | Менять название, описание, имена 12 героев |
| **Карты** | Добавлять/редактировать/удалять карты |
| **Правила** | Редактировать блоки быстрого старта и подробные правила |
| **Об игре** | Менять историю и концепцию |
| **Настройки** | Менять цвета карт, резервная копия |

### Главная страница (index.html)
- 100vh видео фон на весь экран
- Красивая типография с тенью
- 12 героев в 4x3 сетке
- Статистика (12 героев, 88 карт, 2-6 игроков, 20-40 минут)
- Все блоки имеют красивые анимации при скроллинге

### Страница правил (rules.html)
- Блоки кратких правил (редактируются в админке)
- Подробные правила с оглавлением
- Все блоки имеют анимации при скроллинге

## 🔧 Полезные команды

### Скопировать блок правил (в админке)
1. Отредактируйте блок 1
2. Запомните значения (иконка, заголовок, текст)
3. Нажмите "➕ Добавить новый блок"
4. Вставьте те же значения в новый блок
5. Нажмите "💾 Сохранить блок"

### Добавить фоновое изображение к блоку
1. Откройте админку → Правила → Редактировать блок
2. В поле "Фоновое изображение (URL)" вставьте ссылку
3. Ссылка должна быть полная: `https://example.com/image.jpg`
4. Сохраните блок

### Экспортировать данные
1. Откройте админку → Настройки
2. Нажмите "📥 Скачать резервную копию"
3. Сохранится файл `game_data.json`

### Загрузить резервную копию
1. Откройте админку → Настройки
2. Нажмите "📤 Загрузить резервную копию"
3. Выберите файл `game_data.json`

---

## ⚠️ Важные заметки

- Все данные хранятся в `localStorage` браузера
- При очистке истории браузера - данные удалятся!
- Регулярно делайте резервные копии (экспорт)
- Видео должно быть в формате MP4 для совместимости
