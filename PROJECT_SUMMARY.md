## 🎓 Информация для разработчиков

### Как добавить новую страницу?
1. Создайте `new-page.html`
2. Скопируйте структуру из `index.html`
3. Обновите навигацию
4. Добавьте CSS для новой страницы в `styles.css`

### Как модифицировать карты?
1. Откройте `js/data.js`
2. Найдите карту по ID
3. Измените нужные свойства
4. Сохраните файл

### Как добавить новый фильтр?
1. В `js/data.js` создайте функцию фильтрации
2. В `js/cards.js` добавьте обработчик события
3. В `cards.html` добавьте UI элемент
